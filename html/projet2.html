Colmar Academy : Une Plateforme Éducative Responsive pour l'Apprentissage en Ligne

Colmar Academy est un projet de développement web personnel réalisé dans le cadre de ma formation autodidacte via Codecademy, suivant le cours "Build a Website with HTML, CSS, and GitHub Pages". Ce projet visait à mettre en pratique mes compétences en intégration web, en créant un site responsive à partir d'une maquette fournie.

  • Maîtriser l'intégration HTML/CSS : Structurer le contenu de manière sémantique avec HTML et appliquer un style précis et cohérent avec CSS.
  • Développer un site web responsive : Adapter l'affichage du site à différentes tailles d'écran, assurant une expérience utilisateur optimale sur tous les appareils.
  • Utiliser les grilles et les flexbox : Mettre en œuvre des techniques de mise en page modernes pour créer une interface flexible et dynamique.
  • Travailler avec GitHub Pages : Déployer le site en ligne, démontrant ma capacité à utiliser des outils de gestion de version et de déploiement.

Colmar Academy est un projet qui témoigne de ma capacité à transformer une maquette en un site web fonctionnel et attrayant. Il met en évidence ma maîtrise des fondamentaux du développement front-end, mon souci du détail et mon engagement à créer des expériences utilisateur de qualité.
